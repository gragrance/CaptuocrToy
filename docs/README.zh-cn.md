# 如何使用？

1. 你首先需要根据以下命令安装**Xcode Command Line Tools** 和 **cocoapod**。
   - ```xcode-select --install```
   - ```sudo gem install -n /usr/local/bin cocoapods``` 

2. 下载源代码
   - ```git clone https://github.com/gragrance/CaptuocrToy.git``` 

3. 按照如下命令构建

   ```
   cd CaptuocrToy
   pod update
   BuildArea/autobuild.sh
   ```
